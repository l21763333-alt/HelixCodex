# ComboScope Codex Flow — 实验总结

基于 Codex SDK 的预测实验自动诊断与优化。输入实验目录 + 目标，自动完成 **诊断 → 计划 → 代码生成 → 训练验证 → 报告**。

---

## 2. 实验效果 (trial_023)

### 优化内容

在 baseline (LightGBM + regression + full ratio) 基础上：

| 修改 | 内容 |
|------|------|
| 目标函数 | regression → **Tweedie** (variance_power=1.2) |
| 分配策略 | full → **recent weighted** (half_life=7d) |
| 可用性特征 | T+2 工作日可用 / 售卖窗口 / 使用窗口 |
| 节假日特征 | 首末中日 / 公休日前夜 / 周末交叉 |
| 活动力特征 | 套餐全局/城市/门店热度 + 生命周期分桶 |
| 分组校准 | weekday × peak × age × heat 分桶残差校准 |
| 反模式修复 | `pd.to_numeric().fillna()` → `numeric_column_or_default()` (8 处) |

### 指标对比 (test 集, package_detail 粒度)

| 指标 | baseline | trial_023 | 变化 |
|------|----------|-----------|------|
| **WAPE** | 0.7009 | **0.6429** | **-8.3%** |
| **Bias** | +0.1695 | **+0.0837** | **-50.6%** |

**决策: KEEP** ✅
